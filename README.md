# Bike Rental Website

A web application that allows customers to rent bikes and accessories online.

## Project Description
This website is designed for customers to browse, rent, and book bikes and related accessories. Users must sign in before renting any items. The system manages user authentication and rental bookings efficiently.

## Technology Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server & IDE:** XAMPP, VS Code  

## Installation & Setup

1. **Download Project Files**  
   - Download the project folder and SQL file from the repository.

2. **Install XAMPP**  
   - Download and install [XAMPP](https://www.apachefriends.org/index.html).  
   - Start **Apache** and **MySQL** from the XAMPP control panel.

3. **Place Project in `htdocs`**  
   - Copy the project folder into the `htdocs` folder of your XAMPP installation.  
     Example: `C:\xampp\htdocs\projectname`

4. **Import Database**  
   - Open `http://localhost/phpmyadmin/` in your browser.  
   - Create a new database and import the provided SQL file.

5. **Run the Project**  
   - Open your browser and go to:  
     ```
     http://localhost/projectname/
     ```
   - Ensure XAMPP is running and Apache + MySQL are on.

## Features
- User registration and login required for all rentals  
- Rent bikes and accessories online  
- Manage bookings efficiently  

## Usage
1. Sign up or log in as a customer.  
2. Browse bikes and accessories.  
3. Select items and complete the rental booking.
